## 📌 Project Overview

The **Grocery Website** is a full-stack web application that allows users to browse grocery products and manage a shopping cart.

This project demonstrates:
- Frontend and Backend integration
- REST API usage
- Deployment using Vercel,render
- Dynamic data fetching

---

## ✨ Features

- 🛍️ View grocery products
- ➕ Add items to cart
- 🧾 Cart stored using Local Storage
- 🔄 Products loaded from backend API
- ⚡ Fast and responsive UI
- 🌐 Live deployed project

## ⚙️ Installation & Setup

```bash
1. Clone the Repository
git clone https://github.com/Vaishnavi74-star/Grocery-website.git
cd Grocery-website
⚙️ Backend Setup
Install backend dependencies
npm install
Start backend server
node server.js

👉 Backend runs on: http://localhost:5000 (or your configured port)

💻 Frontend Setup
Navigate to frontend folder
cd client
Install frontend dependencies
npm install
Start frontend
npm start

👉 Frontend runs on: http://localhost:3000
